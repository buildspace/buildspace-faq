# Expected private key to be an Uint8Array with length 32

## Why is this happening?

<span>
  This happens because you're most probably using your wallet address instead of your private key.
</span>

## Solution

<span>
    You can get your private key by following the guide [here](https://metamask.zendesk.com/hc/en-us/articles/360015289632-How-to-Export-an-Account-Private-Key).
</span>
